# ETH_Hunt
Collison with Ethereum Address

# Requirements
Need to have a text file named "eth_address.txt" containing the ethereum address to collide with.   
This script will run on Python3 using multicpu. 

# Note
Modify (if you want) the eth_address filename or number of cpu in the script before proceeding.

# Run
On Python3 Terminal, run using the command
```
python ETH_Hunt4.py
```
